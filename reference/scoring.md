# Рубрика подсчёта оценки

Оценка — это **взвешенная плотность маркеров минус контр-сигналы**, с поправкой
на объём. Цель — повторяемость, а не магическое число. Считай прозрачно: показывай,
из чего сложилась оценка.

The score is **weighted marker density minus human counter-signals**, adjusted
for length. Show your work.

---

## Шаг 1. Собери сырые баллы маркеров

За каждый **подтверждённый** маркер начисли по весу категории:

| Вес     | Баллы за вхождение | Категории |
|---------|--------------------|-----------|
| Высокий | 8                  | штампы значимости; «не просто… а…»/негативный параллелизм; техно-блоговые штампы; чат-ботовые артефакты; кластер AI-vocabulary (5+ разных слов) |
| Средний | 4                  | деепричастные надстройки; правило трёх; обтекаемые ссылки/хедж-нагромождение; шаблонная структура/концовка |
| Низкий  | 1.5                | типографика: тире, жирный, эмодзи, Title Case, инлайн-буллеты |

Правила начисления:
- Считай **разные** вхождения, но один и тот же штамп засчитывай максимум 3 раза
  (повтор одного оборота — это 1 привычка, а не 10 улик).
- Чат-ботовый артефакт (категория 8) — сразу +25 к сырому баллу: это почти прямая улика.
- Кластер AI-vocabulary засчитывается один раз как «высокий», даже если слов больше пяти.

Сумму назови `RAW`.

---

## Шаг 2. Нормируй на объём

ИИ-слоп — это про **плотность**. Длинный текст накопит маркеры просто за счёт длины.

```
DENSITY = RAW / (слов / 1000)      // баллы на 1000 слов
```

Базовая оценка по плотности (мягкая сигмоида в виде порогов):

| DENSITY (баллов/1000 слов) | base |
|----------------------------|------|
| 0–8                        | 10   |
| 8–18                       | 30   |
| 18–32                      | 50   |
| 32–50                      | 68   |
| 50–75                      | 82   |
| 75+                        | 92   |

---

## Шаг 3. Вычти контр-сигналы (человеческий шум)

За каждый класс контр-сигналов из `human-signals.md`, реально присутствующий в тексте:

| Контр-сигнал | минус |
|--------------|-------|
| конкретный личный опыт (числа/баги/версии из жизни) | −18 |
| шум/небрежность (опечатки, сленг, мат, обрывы) | −14 |
| мнение/позиция без хеджа, спор, сарказм | −10 |
| диалоговость площадки (обращения, цитаты собеседника) | −10 |
| неровный ритм (высокая дисперсия длины) | −8 |

Минус начисляется один раз за класс (не за каждое вхождение). Нижняя граница после
вычета — 3, верхняя — 97.

```
SCORE = clamp(base − Σ(контр-сигналы) + бонус_структуры, 3, 97)
```

`бонус_структуры` (+, до +10 суммарно): низкая дисперсия длины предложений (+4);
все абзацы начаты со связки (+3); шаблонные разделы Введение/Заключение/Перспективы (+3).

---

## Шаг 4. Поправка на объём → УВЕРЕННОСТЬ

| Слов | Уверенность | Примечание |
|------|-------------|------------|
| < 120  | низкая   | 2–3 маркера на таком объёме — не приговор; не ставь выше «Смешанный» без чат-ботовых артефактов |
| 120–600 | средняя | рабочий диапазон |
| > 600   | высокая  | хватает данных на статистику ритма |

При низкой уверенности смягчай вердикт на одну ступень вниз.

---

## Шаг 5. Вердикт по оценке

| SCORE  | Вердикт |
|--------|---------|
| 0–20   | Человек |
| 21–40  | Скорее человек |
| 41–60  | Смешанный / отредактированный ИИ |
| 61–80  | Скорее ИИ |
| 81–100 | Почти точно ИИ |

---

## Защита от ложных срабатываний (обязательно)

- **Чистый, но конкретный** текст без штампов значимости и с живыми деталями —
  это хороший человек, а не ИИ. Высокая грамотность ≠ генерация.
- **Сухой научный/технический** стиль штрафуй только за *штампы значимости и
  воду*, а не за формальность как таковую.
- **Перевод** живого текста часто ровный и «иностранный» по ритму — склоняйся к
  «Смешанный», не к «Почти точно ИИ».
- Всегда называй 1–2 решающие улики. Если их нет, а есть только типографика —
  итог не выше «Скорее человек».

## Мини-пример расчёта

Текст 300 слов: «важно отметить» (8), «не просто…а…» (8), «под капотом» ×2 (16),
правило трёх ×1 (4), эмодзи-заголовки (1.5). RAW = 37.5.
DENSITY = 37.5 / 0.3 = 125 → base = 92. Контр-сигналов нет. Структура ровная (+4).
SCORE = clamp(96, 3, 97) = 96. Слов 300 → уверенность средняя. **Вердикт: Почти
точно ИИ (96/100), уверенность средняя.** Решающие улики: «не просто…а…» + двойной
«под капотом» при нулевой конкретике.
