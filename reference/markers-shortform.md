# Короткие и платформенные форматы — принципы (RU + EN)

YouTube (заголовки / сценарии / комменты), соцсети (LinkedIn / X), форумы. Опознавай
**каркас и приём**, не фразу. В каждом жанре свой регистр (см. `rewrite.md`).

## YouTube-заголовки
ИИ-приём: разрыв любопытства + стек пауэр-слов + драм-тег в скобках + ALL-CAPS-эмфаза
одного слова + круглое число листикла, всё в одном заголовке. Человек коммитится в
ОДИН угол; точные/не-круглые числа, внутренние отсылки, анти-кликбейт-честность.

## YouTube-сценарии / войсовер
ИИ-приём: целый шаблонный каркас — холодное вступление → стол-транзишн → призыв
подписаться → риторический разворот к зрителю → искусственный саспенс → ровный
sign-off. Человек этот каркас мнёт или пропускает: дисфлюэнсия в монтаже,
самопоправки, таймкоды, реальные имена/цены, региональный сленг.

## YouTube / видео-комменты
ИИ-приём: грамматически полное предложение + общий позитив/похвала + ноль конкретики;
ностальгия-бейт, алгоритм-бейт, чистое прозаическое резюме видео, коммент-список с
жирными заголовками. Человек: фрагментарно, реакция на конкретный момент, сленг,
несогласие, одно слово или один эмодзи.

## Reddit / форум-комменты
ИИ-приём: дипломатично-хеджированный зачин + обе-стороны-изм + идеальное
форматирование + helpdesk-вежливость и подпись-CTA в casual-треде. Человек: резкое
мнение, сарказм, мат, анекдот с грязной конкретикой, одна строка, без подписи.

## Соцсети (LinkedIn / X)
ИИ-приём: вся арка — humble-brag-хук → broetry (однострочные абзацы) → тройка уроков →
бейт-закрывашка (вопрос ради вовлечения). Декоративные эмодзи-буллеты под «вот N
уроков». Человек: реальные имена/числа/скриншот, конкретный провал без аккуратной
морали, нормальное абзацирование, отсутствие CTA.

## Кросс-жанровые эскалаторы и слабые сигналы

Повышают «иишность» в любом коротком формате: правило трёх; негативный параллелизм;
«в современном мире»-зачин; декоративные эмодзи-буллеты; идеальная форма в неряшливой
площадке; обе-стороны-изм; бейт-вопрос последней строкой; канцелярит в casual-тексте.

Слабые/спорные (только в связке, не в одиночку): длинное тире само по себе; «ёлочки» /
curly quotes (бывают дефолтом ввода); одно круглое число.

> The same devices apply in English (clickbait stacks, "like and subscribe"
> scaffolds, generic-praise comment slop, LinkedIn announce-arc); recognise the
> scaffold, not the phrase. When cleaning, break the scaffold, not just the words.
