# Changelog

Все заметные изменения скилла. Формат — по убыванию версий.
All notable changes, newest first.

## 1.4.0
- CI: GitHub Actions (`.github/workflows/ci.yml`) гоняет `validate.mjs` + `eval/run.mjs` на каждый push/PR.
- Добавлен `validate.mjs` — самопроверка SKILL.md по спецификации Agent Skills (без Python; `node validate.mjs`).
- Добавлен `scripts/scan.mjs` — пакетный скан-триаж: считает плотность маркеров по файлам/папке.
- Добавлен `scripts/markers.mjs` — общий набор regex-маркеров и эвристика band'а для инструментов.
- Добавлен eval-харнесс: `eval/cases.json` (размеченные сниппеты) + `eval/run.mjs` (регрессия по полосам вердикта).
- Расширены маркеры (RU + EN): «стоит упомянуть», «нельзя обойти стороной», "when it comes to", "in conclusion", "let's dive/explore".
- Новые примеры-эталоны: перевод → «смешанный», очень короткий текст → низкая уверенность.
- `.gitattributes` (eol=lf) — убирает LF/CRLF-шум.

## 1.3.2
- Frontmatter приведён к спецификации Agent Skills: описание ужато до ~920 символов (лимит 1024),
  `version` перенесён в `metadata`, `allowed-tools` — строкой через пробел, добавлены `license` и `compatibility`.

## 1.3.1
- Новый маркер (RU + EN): энциклопедические уточнения-глоссы в скобках — «(в обиходе „X")»,
  «(он же X)», "(commonly known as X)". Учитывает регистр.

## 1.3.0
- Чистка стала регистро-зависимой: скилл сначала определяет регистр (официальный / пресс / блог /
  форум) и переписывает в его рамках. Нормализация жаргона для формальных текстов («выкатила» →
  «представила»), двусторонняя подгонка под регистр.

## 1.2.0
- Переименование в `ai-slop-cleaner-en-ru`: акцент на чистку ИИ; имя помечено как двуязычное.
- Английское описание и раздел в README; позже README разделён на `README.md` (EN) и `README.ru.md` (RU).

## 1.1.0
- Правило про эмодзи: декоративные ИИ-маркеры (🚀💡✅ в заголовках/буллетах) — улика и вырезаются;
  живые реактивные эмодзи внутри фраз — контр-сигнал за человека.
- Оценка переведена с балльной арифметики на взвешенное суждение; формат ответа стал свободным.

## 1.0.0
- Первая версия: детект + чистка ИИ-текста (RU + EN). Маркеры, контр-сигналы, ориентиры оценки,
  режим чистки, размеченные примеры. Заземление: Wikipedia «Signs of AI writing» и рус. «Признаки
  сгенерированности текста».
